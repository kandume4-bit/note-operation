# title-rules.md — 伸びるタイトルの設計図

記事の価値は中身で決まるが、**読まれるかどうかはタイトルで決まる**。
スルーされたタイトルは存在しないのと同じ。ここはシステムの心臓部として毎回厳守する。

---

## 基本姿勢（2つの絶対原則）

1. **差別化を常に意識する。** どこかで見た型・ありきたりな言い回しは捨てる。「またこれか」と思われた時点で負け。
2. **「思わず開いて読みたくなるか」だけで判断する。** きれいにまとまっていても、開きたくならなければ無価値。

---

## 毎回守る7つのルール

1. **最低7案出す（5型以上をまたぐ）。** 各案に「なぜ目を引くか」を一言添える。出した7案を**4基準（クリック力／具体性／検索SNS／信頼）×各10点＝40点満点**で採点し、**合計点が最高の1案を採用**。最後に**選んだ理由**と別案上位3つもメモに残す。同じ中身でも感情・物語・問いかけでフレームした案を優先する。
2. **具体性を必ず入れる**（数字・期間・固有名詞・対象読者のいずれか）。
   - 「投資の話」→ ◯「月3万円・3年で配当が育った銘柄の選び方」
3. **具体的な数字を必ず盛り込む**（金額・期間・件数・割合）。例「3つのコツ」「14日で」「年利4%」「30分早く寝る」。
   - ただし**根拠のある現実的な数字だけ**。誇張・でっち上げは厳禁。
4. **読者の悩み・欲求を一言で射抜く。** 冒頭3行と矛盾させない。**釣りで終わらせない**。
5. **意外性・新しい切り口を1つ入れる**（常識への反論／見落とされがちな視点／ビフォーアフター）。
6. **検索とSNSの両取り。** 検索向けの具体キーワード＋SNS向けの感情・好奇心を1本に同居させる。
7. **快眠連載は「Day○/14」を入れつつ、その日単体でも読みたくなる**タイトルにする。

---

## タイトルの型（差別化のための引き出し）

ありきたりを避けるため、毎回違う型から選ぶ。型を固定化させない。

| 型 | 構造 | 例 |
|----|------|----|
| **数字×ベネフィット×具体** | 数字＋得られる結果＋誰の何 | 月3万円から始める高配当株、最初の1銘柄の選び方 |
| **常識への反論** | 「実は◯◯はやめていい」 | 早寝はしなくていい。睡眠が変わるのは「起きる時間」だった |
| **ビフォーアフター** | 過去の失敗→今 | 含み損120万円で学んだ、株を買う前に決めるべき3つのこと |
| **失敗・自白系** | 「やらかした」告白 | 新NISAで最初にやった3つの失敗（全部やり直したい） |
| **ギャップ・意外な主語** | 意外な肩書き×テーマ | 運送業20年の運行管理者が、夜勤明けでも7時間眠れた方法 |
| **問いかけ・名指し** | 読者を名指しする問い | 「寝ても疲れが取れない」人がまず削るべきは“朝の30分”です |
| **期間チャレンジ** | ○日でどう変わるか | 14日で寝つきが変わる。Day3「朝の光」を浴びるだけの話 |
| **損失回避** | 「知らないと損」 | 知らずに年1万円損してた、配当金にかかる税金の話 |

> 型はあくまで発想の起点。**型に当てはめただけの“それっぽい”タイトルは避ける**。最後は「自分なら開くか」で判断。

## 良い例（なぜ強いか付き）

- **「月3万円・3年で配当が育った銘柄の選び方」**
  → 金額・期間が具体的で、再現できそうな実感がある。
- **「早寝はしなくていい。睡眠が変わるのは“起きる時間”だった」**
  → 常識への反論＋意外性。続きが気になる。
- **「含み損120万円で学んだ、株を買う前に決めるべき3つのこと」**
  → 生々しい数字＋失敗の自白で信頼と好奇心を同時に取る。
- **「Day3/14｜“朝の光15分”だけで寝つきが変わる理由」**
  → 連載番号を入れつつ、その日単体でも具体的で読みたくなる。

## NG例（埋もれる・ありがち・避けるべき言い回し）

### 抽象的でぼんやり
- 「投資について考える」「睡眠の大切さ」「お金の話」「健康な生活のために」
- 「〜のススメ」「〜とは？」だけで終わる説明的タイトル

### ありがちで量産されている型（手垢がついている）
- 「〜する5つの方法」だけ（数字はあるが意外性・主語がない）
- 「初心者必見！」「保存版」「完全版」「徹底解説」「まとめてみた」
- 「知らないと損する〇〇」を**中身が伴わないのに**使う
- 「人生が変わる」「最強の」「神〇〇」など中身不一致の大げさワード

### 煽り・釣り（開かれても裏切る）
- 「衝撃の事実」「ヤバい」「9割が知らない」（根拠なし）
- 数字をでっち上げる／非現実的な利回り・即効性をうたう

### 無難すぎて埋もれる
- 体言止めの単語並べ「新NISA 高配当 銘柄」（検索だけ狙ってSNSで死ぬ）
- 感情も具体もない「今日の株式市場について」

## 実データで効いた型（2026-06-19・週間ダッシュボードより／随時更新）

実際のPV・スキから判明した傾向。タイトル生成時はこれを踏まえる。

- **ビューを稼ぐ＝問いかけ型「〜と思っていませんか？？」**（常識をゆさぶる問い）。
  - 実績：「ハーブティーは全部ノンカフェイン、と思っていませんか？？」53ビュー（週間最多）／「出来高、なんとなく見過ごしていませんか？」42／「自己資本比率40%、高ければ安心と思っていませんか？」「ナンピン買い、こんな勘違いしてませんか？」も上位。
- **スキ（共感）を稼ぐ＝具体数字＋落とし穴／メカニズム**。
  - 実績：「6月権利確定まであと10日…たった1つのこと」スキ率40%＋唯一コメント2件／「Day3/14｜朝の光を15分浴びると14時間後に…」スキ率34%。数字で「なるほど」を作ると心が動く。
- **ビューが出てもスキ0になる型に注意**＝お得情報・抽象タイトル。
  - 実績：「コストコでディッキーズ874が約半額！！」40ビューでスキ0／「Day2/14｜何時に寝るか…」28ビューでスキ0。**読まれても共感・気づきがないとスキされない。** noteの読者は商品より「視点」を求めている。
- **結論**：理想は「問いかけで開かせ、具体数字＋落とし穴で共感を取る」を1本に同居させる。詳細データは各カテゴリの pv-log と `logs/published.md` 計測欄。
