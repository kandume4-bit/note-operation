高配当株を「利回り」で選んでいませんか。私が今は「何年増配したか」を先に見る理由


「配当利回り5%」という数字を見て、おっ、お得そうだな、と思って買ったことはありませんか。

私はあります。そして、その「高さ」に何度かだまされてきました。

利回りが高いほど良い株だ、と思っていた時期が私にもありました。でも今は、利回りの数字を見るより先に、別のところを見るようになりました。今日はその話を、今日の相場とからめて一緒に考えたいと思います。


――まず、今日の相場をざっと

今日6月30日(火)の東京市場は続伸しました。日経平均の終値は7万0062円32銭(前日比プラス594円21銭、プラス0.86%)で、3営業日ぶりに7万円台を回復しています。TOPIXも3994.76ポイント(プラス12.76、プラス0.32%)と上昇しました。

(出典:日本経済新聞「東証大引け 日経平均は続伸 7万円台回復、AI・半導体に買い」 https://www.nikkei.com/article/DGXZQOFL3054I0Q6A630C2000000/ )

支えになったのは、前日のアメリカ市場です。6月29日の米国市場では、米とイランが報復の一時停止で合意したと伝わって安心感が広がり、S&P500やナスダックが上昇しました。その流れで、今日の東京でも東京エレクトロンやキオクシアといったAI・半導体関連が買われています。寄り付き直後は一時1100円を超えて上げる場面もありました。

ただ、最後は伸び悩みました。今日は6月最終日であり、四半期末でもあります。運用機関が持ち高を調整する「リバランス」の売りが出やすく、上げ幅をだいぶ削って引けています。ドル円は1ドル=161円台後半と、節目の162円をうかがう円安水準が続いていました。

(出典:外為どっとコム「今日のドル円相場見通し 2026年6月30日」 https://www.gaitame.com/media/entry/2026/06/30/092144 )

派手に動いたのは、今日もやっぱりAI・半導体でした。こういう日は、地味な高配当株は話題になりにくいんですよね。でも私は、こういう日こそ高配当株の話をしておきたいと思っています。理由はあとで書きます。


――そもそも配当利回りって何でしたっけ

念のため言葉から。配当というのは、株を持っているだけで会社から年に1〜2回もらえるお金のことです。そして配当利回りは、「払った株価に対して、1年でどれくらい配当をもらえるか」を割合にしたものです。

たとえば1株2000円の株が年80円の配当を出すなら、80÷2000で利回りは4%。同じ80円でも株価が1600円なら、80÷1600で5%になります。

ここでもう、ひとつ気づいてほしいことがあります。利回りは「配当が増えた」ときだけでなく、「株価が下がった」ときにも上がる、ということです。これが今日いちばん伝えたいことの入り口です。


――「利回りが高い」が、いつも良い話とは限らない

私が昔やった失敗は、まさにここでした。スクリーニングで利回りの高い順に並べて、上のほうにある株を「お得だ」と思って買う。これをやると、ときどき地雷を踏みます。

なぜなら、利回りが妙に高い株のなかには、「業績が悪化して株価が売られた結果、見かけ上の利回りだけが高くなっている」ものが混じっているからです。会社の中身が弱っているのに、過去の配当額で計算するから利回りだけ立派に見える。これを私は勝手に「見かけの高利回り」と呼んでいます。

こういう株は、次の決算で配当そのものを減らす「減配」に踏み切ることがあります。そうなると、配当は減る、株価はさらに下がる、の二重の痛手です。配当目当てで入ったのに、値下がりでそれ以上に損をする。順番が逆になってしまうわけです。

(出典:会社設立のミチシルベ「高配当株をおすすめしない理由とは?危険な銘柄の見抜き方」 https://www.soico.jp/no1/news/securities/4332 )

もうひとつ怖いのが、専門用語で「タコ配」と呼ばれるものです。タコが自分の足を食べる、という例えで、会社が十分に稼げていないのに、無理して配当を出し続けている状態を指します。利益以上に配当を出していれば、いつかは続かなくなる。高い利回りの裏に、この無理が隠れていることがあります。


――だから私は今、「何年増配したか」を先に見ます

では利回りの数字以外に、どこを見るのか。私が先に確認するようになったのは、「その会社が何年連続で配当を増やしてきたか(連続増配)」と、「利益のうち何割を配当に回しているか(配当性向)」の2つです。

連続増配が長い会社は、不景気の年をはさんでも配当を減らさずにこられた、という実績がある会社です。たとえば花王は36期連続で増配を続けていて、これは国内でも最長クラスです。三菱HCキャピタルのように、20年を超えて毎年増配し、利回りもおおむね3〜4%台にある会社もあります。

(出典:ダイヤモンドZAi「連続増配株ランキング ベスト20[2026年最新版]」 https://diamond.jp/zai/articles/-/229803 、同「三菱HCキャピタル、28期連続の増配を発表」 https://diamond.jp/zai/articles/-/1067215 )

念のためはっきり書いておきますが、これは「この銘柄を買いましょう」という話ではありません。あくまで、「長く増配を続けられている会社は、配当を出す体力と意思がある会社だと考えられる」という、見方の例として挙げています。

配当性向のほうは、ざっくり30〜50%くらいに収まっていると、無理なく払えていると見られることが多いようです。逆に100%を超えていたら、さっきのタコ配を疑う材料になります。利益が出ていない年に配当だけ満額、というのは長続きしにくいですよね。

(出典:ダイヤモンドZAi「配当が減らない隠れ増配株ランキング」 https://diamond.jp/zai/articles/-/239295 )


――今日のような日に高配当の話をする理由

最初に「こういう日こそ高配当株の話をしたい」と書きました。理由はこうです。

今日みたいにAI・半導体が一時1100円も上げる日を見ると、つい「自分も値上がりしそうな株で一発」と気持ちが前のめりになります。私もそうです。でも高配当株は本来、毎日の株価で一喜一憂するより、もらえる配当をコツコツ積み上げていく付き合い方が向いています。値動きの派手な日ほど、自分はどっちのつもりで持っているのか、を確かめる良い機会だと思うんです。


――今日の学び(1つだけ)

高配当株は、利回りの「高さ」で選ばない。「その配当を続けられる会社か」を、連続増配の年数と配当性向で先に確かめる。利回りはそのあとで見れば十分です。

正直に言うと、私も昔は利回りランキングの上から買って、何度か痛い目を見ました。だから今は、派手な数字を見たときほど一拍おいて、「これは続くのかな」と自分に聞くようにしています。

あくまで私の一例です。最後はご自身で判断してくださいね。今日もおつかれさまでした。
